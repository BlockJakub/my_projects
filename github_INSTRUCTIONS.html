GitHub Instructions

Follow these steps to manage repositories, download projects, and update them on GitHub:

Switching Between Repositories in VS Code

  1. Open a Repository: Go to File > Open Folder in VS Code. Navigate to the folder of the repository you want to work on and open it.
  2. Use the Source Control Panel: Click on the Source Control icon in the Activity Bar (on the left side of VS Code). If the repository is already initialized, you’ll see the Git controls for that repository.
  3. Switch Between Repositories: If you have multiple repositories open in a workspace, you can switch between them by clicking the dropdown in the Source Control panel.
  4. Close a Repository: To close a repository, simply close the folder in VS Code by going to File > Close Folder.

Downloading a Project from GitHub to VS Code

  1. Clone the Repository:
    • Open the Command Palette in VS Code (Ctrl+Shift+P or Cmd+Shift+P on Mac).
    • Type Git: Clone and select it.
    • Paste the repository URL (e.g., https://github.com/username/repository.git) and press Enter.
    • Choose a local folder where the repository will be cloned.
  2. Open the Cloned Repository: After cloning, VS Code will prompt you to open the repository. Click Open.
  3. Install Dependencies (if applicable):
    • If the project has a package.json (Node.js), requirements.txt (Python), or similar file, open the terminal in VS Code (Ctrl+`) and run the appropriate command:
    • For Node.js: npm install
    • For Python: pip install -r requirements.txt
  4. Start Coding: You can now edit files, add new features, and commit changes.

Updating the Project on GitHub

  1. Stage Changes:
    • In the Source Control panel, you’ll see a list of modified files.
    • Click the + icon next to each file to stage it, or click the + icon at the top to stage all changes.
  2. Commit Changes:
    • Enter a commit message in the text box at the top of the Source Control panel.
    • Click the checkmark icon to commit the changes.
  3. Push Changes to GitHub:
    • Open the terminal in VS Code (Ctrl+`) and run:
    • git push
    • Alternatively, click the Sync Changes button in the bottom-left corner of VS Code.

Summary of Git Commands

Action Command Description
Initialize Repository git init Initialize a new Git repository in the current directory.
Add Files git add . Stage all changes in the current directory for the next commit.
Commit Changes git commit -m "Your commit message" Commit the staged changes with a descriptive message.
Set Remote Repository git remote add origin https://github.com/username/repository.git Link your local repository to a remote repository on GitHub.
Push to GitHub git push -u origin main Push your changes to the main branch of the remote repository.
Clone Repository git clone https://github.com/username/repository.git Download a repository from GitHub to your local machine.
Check Status git status View the status of your working directory and staged changes.
Create New Branch git branch branch_name Create a new branch in your repository.
Switch Branch git checkout branch_name Switch to a different branch in your repository.
Merge Branch git merge branch_name Merge a branch into the current branch.
Pull Changes git pull origin main Fetch and merge changes from the remote repository.
View Commit History git log View the commit history of the repository.
